# StockTell 视觉设计系统规范

> 目的:统一视觉语言,后续 UI 迭代以此为依据。
> 定位:**「认识我、说真话、敢亮战绩、会提醒」的盯盘搭子** —— 视觉应传达**克制、可信、编辑感**,而不是"信息很密的焦虑工具"。

## 一、现状诊断(2026-06,首页今日简报)

**做得好(保留):** 三段式内容模型(为什么动 / 受益 / 散户怎么想)、和我相关↔其他分区、资金面折叠、情绪条概念。

**核心问题:**
1. **色彩语义冲突** —— 红色同时表示:涨(红涨绿跌)、高影响、链接、强调,一色多义,无法快速解码。
2. **琥珀泛滥** —— 早报/散户怎么想/资金面/过期 banner/自选高亮 全是琥珀填充块,无重点且显廉价(琥珀=警告暗示)。
3. **卡片冗长** —— 每卡含整段"散户怎么想",一屏看不下两条,滚动疲劳。
4. **层级偏弱** —— 分区标题是小灰字;两区卡片几乎同款;标题/正文/标签字阶扁平。
5. **CTA 噪音** —— "想更深入?让 StockTell 帮你解读这条→"每卡重复。
6. **情绪条未可视化** —— 一串拥挤小字数字,没用上"一眼看懂"的机会。

## 二、设计规范(迭代依据)

### 1. 色彩语义(铁律:一色一义)
- **涨跌**:仅**数字/价格**用 红涨绿跌(`text-rose-600` / `text-emerald-600`)。其它地方不得用红/绿表意。
- **影响等级**:不用饱和圆点 + 红标题。改为**卡内左侧内嵌圆角细色条**(上下留白、两端圆角,不与卡片边框相接 —— 避免"半截彩色边框"在圆角处割裂)+ **描边小标签**(不填色),低饱和:
  - 高 `rose`、中 `amber`、低 `gray`,但都走浅色/描边,不抢主体。
- **品牌强调色**:全站统一一个克制主色(建议 `indigo/slate` 系)用于链接、CTA、"你的"标识。
- **琥珀**:降级为**仅"自选/早报"专属**,且用极浅 tint(`amber-50/40`)+ 左标尺,不整块填充。

### 2. 字阶(Type scale)
- 卡片标题:`text-[15px]` / `font-semibold` / `text-gray-900`
- 正文:`text-sm` / 行高 `leading-relaxed`(1.7)/ `text-gray-700`
- meta/标签:`text-xs` / `text-gray-400`
- 分区标题:`text-base` / `font-semibold` / `text-gray-800`(明显大于正文)

### 3. 间距与容器
- 多留白、**少边框线**:卡片优先"无框 + 充足内距 + 极细分隔",而非每卡全包 `border`。
- 卡片内距 `p-4`,卡间距 `space-y-3`;分区间距 `space-y-8`。

### 4. 卡片规范(Briefing Card)
结构:`[左色条] 影响描边标签 · 标题` → 一行 meta(为什么动 / 受益A股)→ **散户怎么想(默认 2 行截断 + 展开)**。
- "散户怎么想"**默认 `line-clamp-2` + 「展开」**,点开看全文 → 卡片高度减半、可扫描。
- "受益A股"压成一行 chip;命中自选的 chip 高亮(琥珀)。
- CTA 不每卡重复:**整卡可点 / 或悬停出现轻图标**。

### 5. 分区层级
- 「和我相关」:整区给极浅品牌底,标题加大 + 自选数 badge,和「其他市场动态」明确拉开。
- 自选命中的卡:左色条/标识区分,不靠整圈琥珀 ring。

### 6. 情绪条(可视化)
- 涨跌家数 → **红绿堆叠条**(一眼看多空比);主数字(均涨跌)放大;主力净流入用箭头/色块。
- 从"读数字"变"看一眼就懂"。

## ★ Design System(已确认:品牌色 = indigo,token 进 `tailwind.config`)

> 实现机制:`tailwind.config.ts` 的 `theme.extend` 定义命名 token;`globals.css` 写 `--brand/--canvas` CSS 变量(为深色模式留口)。新增不破坏现有类,迁移分批做。

### ✅ 最终视觉(已确认,全员遵守 —— 以 `prototype/home.html` 为基准)
扁平 / 无边框 / 用光影与底色做层级,不用线条。
- **卡片**:`bg-surface`(白)+ `rounded-xl` + `shadow-sm`,**不要 border、不要左色条**。靠阴影与面板底色区隔。
- **模块面板**:两个区块同款容器 `rounded-2xl p-4`,**只用底色区分、不要 ring/border**:
  - 和我相关 = `bg-brand-50/40`(你的专区);其他市场动态 = `bg-gray-100/50`(大盘)。
- **徽章(全部扁平)**:浅填充 + 彩字,**无描边、无阴影**。
  - 影响:高 `bg-rose-50 text-rose-600` / 中 `bg-amber-50 text-amber-700` / 低 `bg-emerald-50 text-emerald-600`。
  - 自选 chip:`bg-amber-100 text-amber-800`(★);情绪 mood:同上扁平。
- **"散户怎么想"**:`bg-gray-50/70` 浅灰填充块,**无左标尺/边框**。
- **早报 / 过期提示**:`bg-amber-50` 纯底,无边框(amber = 个人/提示语义)。
- **数据可视化**:涨跌家数/主力净流入用红绿小条(`bg-rose-400`/`bg-emerald-400`);红涨绿跌只给数据。
- **唯一保留的线**:页面 header 底部 `border-b`(结构分隔);组件内部必要的极细 divider 可保留。

### 颜色 token(一主一辅 + 两类语义)
| token | 值 | 用途 | 铁律 |
|---|---|---|---|
| `ink` | `#1a1d24` | 主文字/主按钮(墨色) | 主操作 |
| `brand`(=indigo 50–900) | indigo | 链接 / focus / 你的专区 / 导航当前项 / 选中 | **唯一品牌强调色**,取代散落的 blue/indigo |
| `canvas` | `#f7f8fa` | 页面底 | 取代硬编码 #f7f8fa |
| `surface` | `#ffffff` | 白卡 | |
| `amber-*` | — | "你的/个人"(早报、自选)| 稀疏,**不当品牌色** |
| `rose / emerald` | — | **仅数据**:涨跌、影响等级(低饱和) | 别处禁用 |

### 字阶 token(`fontSize`,只含 size+lineHeight,字重另用 `font-*`)
| token | px / line-height | 用途 |
|---|---|---|
| `meta` | 12 / 1.4 | 标签、时间、辅助(取代 `text-[11px]`) |
| `body` | 14 / 1.7 | 正文 |
| `title` | 15 / 1.45 | 卡片标题(取代 `text-[15px]`) |
| `h2` | 17 / 1.35 | 分区标题 |
| `h1` | 22 / 1.3 | 页面标题 |
| `display` | 24 / 1.2 | 命中率/大数字 |
> 保留 Tailwind 原生 `text-xs/sm`(≈meta/body);新代码优先用命名 token,旧的逐步收敛。

### 圆角 / 间距 / 表面
- 圆角:`chip`=8(`rounded-lg`)、`card`=12(`rounded-xl`)、`panel`=16(`rounded-2xl`)——三档。
- 间距节奏:卡内 `p-4`、卡间 `space-y-3`、分区间 `space-y-8`。
- 表面层级:`canvas`(页底)→ `surface`(白卡)→ `surface-muted`(浅灰块,取代散落 `bg-gray-50`)。

### `tailwind.config` 片段(实现参考)
```ts
import colors from "tailwindcss/colors";
theme: {
  extend: {
    colors: { brand: colors.indigo, ink: "#1a1d24", canvas: "#f7f8fa", surface: "#ffffff" },
    fontSize: {
      meta:  ["0.75rem",   { lineHeight: "1.4" }],
      body:  ["0.875rem",  { lineHeight: "1.7" }],
      title: ["0.9375rem", { lineHeight: "1.45" }],
      h2:    ["1.0625rem", { lineHeight: "1.35" }],
      h1:    ["1.375rem",  { lineHeight: "1.3" }],
      display:["1.5rem",   { lineHeight: "1.2" }],
    },
  },
}
```

### 迁移范围(确认原型效果后执行)
`blue-*`/`indigo-*`→`brand-*`、`#f7f8fa`→`bg-canvas`、`#1a1d24`→`text-ink`、`text-[11px]`→`text-meta`、`text-[15px]`→`text-title`;`text-xs/sm`、`rounded-*` 作约定逐步收敛,不一次性全量改(降低与维护者并行的撞车风险)。

## 三、迭代节奏
- **快赢(体感提升最大)**:降琥珀 + 收红(色彩语义)、卡片去边框 + "散户怎么想"折叠、CTA 收敛。
- **中期**:字阶/间距 token 化、情绪条可视化、分区层级强化。
- **勿动**:三段式内容模型、和我相关/其他分区、资金面/战绩既有逻辑。
